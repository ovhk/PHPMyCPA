PHPMyCPA est un logiciel de galerie de cartes postales anciennes pour Internet écrit en php,
il utilise MySQL, et un moteur de template ModeliXe (version beta 0.6.6).
Libre à vous de le télécharger, le modifier, le redistribuer gratuitement.
Si vous avez de nouvelles fonctionnalités à proposer, merci de m'en faire part.

---- description des fichiers ----
- admin
   - fonctions.inc.php
   - index.php
   - table.sql
- originaux
	   repertoire contenant les images originales .jpg
	   elles seront nommés de la façon suivante :
	   PAYS__DEPARTEMENT__VILLE__DESCRITION_DE_LA_PHOTOS__ANNEE.jpg
- big
- small
